    🔗 디스코드 연동

    조직에 새 공지 · 경고 · 휴가 신청/처리 · 가입 신청이 생기면 디스코드 채널로 자동 알림이 전송됩니다.

    1. 디스코드에서 알림 받을 채널 → 편집(톱니) → 연동 → 웹후크
    2. 새 웹후크 만들고 웹후크 URL 복사
    3. 아래에 붙여넣고 저장 → 테스트 전송으로 확인

    🔑 디스코드 로그인 (OAuth)

    디스코드 계정으로 로그인하고, 본인 디스코드를 계정에 연동할 수 있습니다. (역할 자동 동기화의 전제)

    1. 디스코드 개발자 포털 → New Application
    2. OAuth2 → Client ID / Client Secret 을 Render Environment에 DISCORD_CLIENT_ID, DISCORD_CLIENT_SECRET로 등록
    3. OAuth2 → Redirects에 아래 Redirect URI를 그대로 추가

    🎖️ 역할 자동 동기화 (봇)

    앱에서 직급이 바뀌면 디스코드 역할도 자동으로 변경됩니다. (봇 상시 연결 불필요 — REST 호출)

    1. 개발자 포털 → Bot → 토큰을 Render Environment에 DISCORD_BOT_TOKEN으로 등록
    2. 봇을 서버에 초대(권한: 역할 관리) 후, 서버 설정에서 봇 역할을 관리할 역할들보다 위로 올리기
    3. 아래 서버 ID와 직급별 역할 ID를 입력하고 저장 (디스코드 개발자 모드 켜고 우클릭 → ID 복사)
